## Main Project

### [RDC Drug Search Platform](https://github.com/crushh/drug)

Next.js + TypeScript + MySQL application for searching and displaying radioligand drug conjugate data.

What it demonstrates:

- Searchable web application built with Next.js App Router and React
- API routes for RDC search, status filtering, detail pages, chemical entity lookup, and reference data
- Normalized MySQL schema for drugs, targets, indications, chemical entities, in vitro data, references, and many-to-many relationships
- SQL query logic for fuzzy search, filtering, sorting, pagination, joins, and detail expansion
- CSV templates, validation reports, and import scripts for structured data maintenance
- GitHub Actions workflows for code deployment and database CSV patch flow

Tech: `Next.js` `React` `TypeScript` `MySQL` `mysql2` `API Routes` `SQL` `GitHub Actions`
