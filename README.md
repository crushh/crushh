## Main Portfolio Project

### RDC Drug Search Platform

Next.js + TypeScript + MySQL application for searching and displaying radioligand drug conjugate data.

Repository: https://github.com/crushh/drug

What this project demonstrates:

- Built a searchable web application with Next.js App Router and React
- Implemented API routes for RDC search, status filtering, detail pages, chemical entity lookup, and reference data
- Designed a normalized MySQL schema for drugs, targets, indications, chemical entities, in vitro data, references, and many-to-many relationships
- Wrote SQL query logic for fuzzy search, filtering, sorting, pagination, joins, and detail expansion
- Added CSV templates, validation reports, and import scripts for structured data maintenance
- Included GitHub Actions workflows for code deployment and database CSV patch flow

Tech stack:

`Next.js` `React` `TypeScript` `MySQL` `mysql2` `API Routes` `SQL` `GitHub Actions`
